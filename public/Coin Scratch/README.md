# Coin Scratch ASMR

A premium ASMR-inspired coin scratching web experience built using HTML, CSS, and JavaScript.  
This project recreates the relaxing feeling of scratching a rusted coin surface to reveal a calming reward animation underneath.

The experience includes:

- realistic scratch interaction
- soothing metallic scratching audio
- premium 3D coin design
- calming pastel ASMR UI
- smooth reward reveal animations

Perfect for relaxation-focused web experiences.

## Features

- ASMR-style scratching experience
- Premium 3D coin UI
- Realistic scratch reveal effect
- Heart reward animation
- Smooth “You Did It” reveal transition
- Procedural metallic scratching audio
- Calm pastel luxury background
- Smooth UI animations and glow effects
- Restore/Reset coin functionality
- Responsive centered layout

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Canvas API
- Web Audio API
